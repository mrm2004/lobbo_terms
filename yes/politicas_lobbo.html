Perodo de conservao
Os Dados Pessoais sero processados e armazenados pelo tempo que for necessrio para as finalidades para
as quais forem coletados.
Portanto:
Os Dados Pessoais coletados para as finalidades relacionadas com a execuo de um contrato entre o
Proprietrio e o Usurio sero conservados at que tal contrato tenha sido completamente cumprido.
Os Dados Pessoais coletados para as finalidades relacionadas com os legtimos interesses do
Proprietrio sero conservados pelo tempo que for necessrio para cumprir tais finalidades. Os Usurios
podero obter informaes especficas sobre os interesses legtimos perseguidos pelo Proprietrio
dentro das sees pertinentes deste documento ou entrando em contato com o Proprietrio.
O Proprietrio poder ter a permisso de conservar os Dados Pessoais por um prazo maior sempre que o
Usurio tiver dado a sua autorizao para tal processamento, enquanto tal autorizao no tiver sido
retirada. Alm disso, o Proprietrio poder ficar obrigado a conservar os Dados Pessoais por um prazo maior
em todas as ocasies em que estiver obrigado a faz-lo para o cumprimento de uma obrigao jurdica ou
em cumprimento de um mandato de uma autoridade.
Assim que o prazo de conservao vencer os Dados Pessoais sero apagados. Desta forma o direito de
acessar, o direito de apagar, o direito de corrigir e o direito portabilidade dos dados no podero ter o seu
cumprimento exigido aps o vencimento do prazo de conservao.
